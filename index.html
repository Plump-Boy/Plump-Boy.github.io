手把手教你搭建个人博客
更新于 2019-11-19
博客相关

之前看了豆哥的博客搭建教学，自己也想搭一个玩玩，奈何个人技术是在太差，看不懂豆哥那个的操作流程，又去网上找教程结果又踩坑了，最终兜兜转转一天的功夫才搞好，现在与大家分享一下流程，也让大家少走弯路。

1.准备工作

1.注册一个GitHub账号

2.下载并安装node.js(https://nodejs.org/en/)

3.下载并安装git(https://git-scm.com/)

4.如果你想拥有一个属于自己的域名的话，提前去阿里云上面进行实名认证并且买一个域名（一般一年三四十块钱）

2.配置SSH key

此步骤可做可不做，GitHub配置SSH Key的目的是为了帮助我们在通过git提交代码是，不需要繁琐的验证过程，简化操作流程。
1.打开git bash（之前下载git后可以找到这个程序），第一次使用要配置一下，输入下列代码

git config --global user.name "xxxxx"
git config --global user.email  "xxxxx@xxxxx.xxx"

2.输入

cd ~/.ssh
ls
ll

检查有没有id_rsa 和 id_rsa.pub文件，如果存在则说明已经有SSH key，如果没有则需要生成一个。

ssh-keygen -t rsa -C "xxxxx@xxxxx.xxx"

输入完后应该会提示一些东西，不用管，敲三下回车。
此时进入C:\Users\你的用户名\.ssh里面，找到id_ras.pub文件并且用记事本打开它，复制里面所有的内容。

这时打开GitHub，点击setting

再点击SSH and GPG keys

再点击New SSH key


最终在key内粘贴之前复制的内容（title随便写），点击Add SSH key即可s。

3.最后进行检查，再git bash中输入

ssh -T git@github.com

运行结果出现类似如下

Hi xxxxx! You've successfully authenticated, but GitHub does not provide shell access.

那么SSH key就已经配置成功了。

3.安装Hexo

1.在你喜欢的地方新建一个文件夹，并且在git bash里面进入这个文件夹（cmd命令请自行百度），输入

npm install -g hexo


接着输入，此处会有些慢，要等待一下。

hexo init


然后输入

hexo s

会启动本地的4000端口，http://localhost:4000 打开这个你就能在本地看到你的博客了。注意在看完之后返回git bash摁ctrl+c关闭。如果没有可以用hexo clean命令清除一下，再hexo g生成一次，再hexo s尝试。当然也有可能是4000端口占用，解除方法百度。

4.把博客推到GitHub库中

1.打开github，点击your repositorise

点击右边的new新建一个库

特别注意！！！库的名称一定要是:你的github用户名.github.io，最后点击create repositorise完成创建。

2.在git bash中输入代码，等待安装插件完成。

cnpm install --save hexo-deployer-git

3.在这个目录下面用记事本打开config.yml这个文件，在最下面deploy那里，全部改成（注意中间的空格！）

deploy；
  type: git
  repo: 你的库的地址
  branch: master

然后保存并退出。
最后在git bash里面输入

hexo d

可能还要输入一次github账号和密码。

4.刷新github的库的页面，会发现多了很多东西，在浏览器里面输入刚刚创建的库名称，我们的博客就可以显示出来了，博客搭建成功！

5.绑定域名

1.打开阿里云控制台，在域名管理里面你的域名后面点击解析

2.点击添加记录
创建两次，记录类型一定要选择CNAME!!主机记录一次www，一次@，记录值就是你的个人博客的地址（去掉https：//其实跟仓库名称是一样的）

确定之后在github库的页面点击setting，

在下面的github pages里面的Custom domain写下你的域名，

这样就可以用域名访问你的博客了。至于后面的更换主题写博客之类的就自己去研究了。

写在最后

现在回过头来写确实感觉步骤十分简单，可就是这样的步骤在我搭建的时候错误也是不断，你现在看到的是一个混杂版，前半部分是一个网上的教程，可惜他后半部分是错的，所以当初在搭建的时候就一直没法成功，后来改成了codesheep的方法搭建后半部分才得以成功。对我感触最大的应该是用百度好处了，在搭建过程中出现的一个又一个问题基本上都能在百度上找到答案（虽然也有些没能解决）。在我看来，理解每一步是在干啥，能收获的比结果更多。

C语言易错点总结
更新于 2019-11-19
C语言

本帖主要汇总我在学习C语言过程中自己出现的一些错误，踩的一些雷区，希望你看完之后能少犯一些错误 ヾ(๑╹◡╹)ﾉ”

  1. 标识符规范：

    标识符必须由字母，下划线或数字组成。
标识符必须以字母或下划线开头。
关键字不可作为标识符。
由于C语言中严格区分大小写，故某一字符大写的标识符仍为合法标识符，例如Int或printF（使用时不建议如此命名标识符）

  1. 整数除以整数结果还是整数，这个结果为不大于实际结果的最大整数，如果想让结果变为小数则可以把其中一个整数变成浮点数再进行运算。求余运算时（%）注意%两边都只能是整型变量。

  2. C语言中不同变量在输入输出时对应不同格式说明符，例如int类型对应%d，flout类型对应%f,double类型对应%lf（输出时也可为%f）等等，在使用时记得对应自己要输出的数据类型使用。

  3. printf语句与scanf语句格式问题
    printf语句格式为printf(“格式控制”,输出列表)；所以下文中x前面没有&，但是scanf语句格式，scanf（”格式控制”，地址列表);所以在下文中x前添加了&。

    int x;
scanf("%d",&x);
printf("%d",x);
return 0;

  4. printf（“%a.bd”,x）;
    在这样的输出语句中，a叫做宽度限定词，作用是制定数据的输出宽度（符号位也计算宽度），若数据实际位数小于a，则左端补空格，若大于a，则按实际位数输出。b的作用则是用来使浮点数输出时保留b位小数（最后一位会四舍五入）。

  5. printf（）中要想打印\,',"由于它们是用于定义字符常量，是printf（）函数的一部分，直接使用会造成混乱，则应该在双引号中如此输入\\,\',\"

  6. 非静态局部变量使用之前要先进行赋值，例如在s=s+1这个语句中，如果s没有进行提前赋值那么出来的结果就是随机数。

    静态局部变量在有static标记时，会默认初始化为0。

    全局变量没有初始化值时会自动赋值0。

  7. 一个等号（=）是赋值运算符，作用是赋值；两个等号（==）是关系运算符，作用是判断相等，若==两边相等则表达式结果为1，否则为0。在判断相等时切记不可只用一个等号。

  8. 在书写诸如if-else，for，while等语句的内嵌语句时，只允许是一条语句，若有多条语句，则应该使用大括号将这些语句组成复合语句，如果不打大括号，程序则只能将第一条语句作为内嵌语句执行。

  9. 字符型变量输入可调用scanf（）和getchar（），输出可调用printf()和putchar（）。需要注意的是getchar()和putchar()只能处理单个字符，即调用一次函数只能输入或输出一个字符。
    特别提示：由于空格和回车也是字符，所以在输入字符时字符间不要加空格，在字符全部输入完之后再回车

  10. 在使用嵌套循环时，一定要明确变量对内外层循环初始化，对外层循环初始化应放在外层for之外，对内层循环初始化应放在外层for之内，内层for之外。

  11. break只能用于switch语句和循环语句

  12. 在下面的例子中，n只能为整型变量和字符型变量，a和b只能为常量表达式，case后面是：（冒号）而不是；（分号），如果case后面没有break，那么它将继续执行下去直到遇见break或者全部执行完毕。

switch（n）
case(a):x=1;break;
case(b):x=2;break;

  1. break语句是跳出整个循环，而continue则是跳过下一语句。

  2. i++与++i的区别：在单独使用时没有区别，都表示i=i+1,
    但是a=++i,表示的是i=i+1；a=i；
    而a=i++，表示的是a=i；i=i+1；
    （此处涉及到运算符的优先级和结合性问题，后面会详细说明）

16.在if和for语句等语句后面不要加分号，例如

错误示例：if(a==0);
正确示例：if(a==0)
错误示例：for(i=0;i<=10;i++);
正确示例：for(i=0;i<=10;i++)

同时注意for语句中每个语句用分号间隔而不是逗号。
特别的，在while语句中，while后面无需加分号，而在do-while语句中while后面要加。

17.数组当中下标是从0开始的，下标的合理取值范围为[0，数组长度-1]，例如一个规定了10个元素的数组a，a[10]是不存在的。它的元素为a[0]-a[9]。

18.在自定义函数中，return只能返回一个函数值，如果存在多个值需要返回主函数，则应该把自定义函数类型设置为void，对于需要返回的值设置全局变量,void函数中可以存在return，但是return后面不能加任何数值，表示自定义函数到此为止。
