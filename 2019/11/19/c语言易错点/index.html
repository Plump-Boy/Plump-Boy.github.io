C语言易错点总结

2019年11月19日 1,386 字 大概 5 分钟 C语言 C语言

本帖主要汇总我在学习C语言过程中自己出现的一些错误，踩的一些雷区，希望你看完之后能少犯一些错误 ヾ(๑╹◡╹)ﾉ”

  1. 标识符规范：

    标识符必须由字母，下划线或数字组成。
标识符必须以字母或下划线开头。
关键字不可作为标识符。
由于C语言中严格区分大小写，故某一字符大写的标识符仍为合法标识符，例如Int或printF（使用时不建议如此命名标识符）

  1. 整数除以整数结果还是整数，这个结果为不大于实际结果的最大整数，如果想让结果变为小数则可以把其中一个整数变成浮点数再进行运算。求余运算时（%）注意%两边都只能是整型变量。

  2. C语言中不同变量在输入输出时对应不同格式说明符，例如int类型对应%d，flout类型对应%f,double类型对应%lf（输出时也可为%f）等等，在使用时记得对应自己要输出的数据类型使用。

  3. printf语句与scanf语句格式问题
    printf语句格式为printf(“格式控制”,输出列表)；所以下文中x前面没有&，但是scanf语句格式，scanf（”格式控制”，地址列表);所以在下文中x前添加了&。

    int x;
scanf("%d",&x);
printf("%d",x);
return 0;

  4. printf（“%a.bd”,x）;
    在这样的输出语句中，a叫做宽度限定词，作用是制定数据的输出宽度（符号位也计算宽度），若数据实际位数小于a，则左端补空格，若大于a，则按实际位数输出。b的作用则是用来使浮点数输出时保留b位小数（最后一位会四舍五入）。

  5. printf（）中要想打印\,',"由于它们是用于定义字符常量，是printf（）函数的一部分，直接使用会造成混乱，则应该在双引号中如此输入\\,\',\"

  6. 非静态局部变量使用之前要先进行赋值，例如在s=s+1这个语句中，如果s没有进行提前赋值那么出来的结果就是随机数。

    静态局部变量在有static标记时，会默认初始化为0。

    全局变量没有初始化值时会自动赋值0。

  7. 一个等号（=）是赋值运算符，作用是赋值；两个等号（==）是关系运算符，作用是判断相等，若==两边相等则表达式结果为1，否则为0。在判断相等时切记不可只用一个等号。

  8. 在书写诸如if-else，for，while等语句的内嵌语句时，只允许是一条语句，若有多条语句，则应该使用大括号将这些语句组成复合语句，如果不打大括号，程序则只能将第一条语句作为内嵌语句执行。

  9. 字符型变量输入可调用scanf（）和getchar（），输出可调用printf()和putchar（）。需要注意的是getchar()和putchar()只能处理单个字符，即调用一次函数只能输入或输出一个字符。
    特别提示：由于空格和回车也是字符，所以在输入字符时字符间不要加空格，在字符全部输入完之后再回车

  10. 在使用嵌套循环时，一定要明确变量对内外层循环初始化，对外层循环初始化应放在外层for之外，对内层循环初始化应放在外层for之内，内层for之外。

  11. break只能用于switch语句和循环语句

  12. 在下面的例子中，n只能为整型变量和字符型变量，a和b只能为常量表达式，case后面是：（冒号）而不是；（分号），如果case后面没有break，那么它将继续执行下去直到遇见break或者全部执行完毕。

    switch（n）
case(a):x=1;break;
case(b):x=2;break;

  13. break语句是跳出整个循环，而continue则是跳过下一语句。

  14. i++与++i的区别：在单独使用时没有区别，都表示i=i+1,
    但是a=++i,表示的是i=i+1；a=i；
    而a=i++，表示的是a=i；i=i+1；
    （此处涉及到运算符的优先级和结合性问题，后面会详细说明）

  15. 在if和for语句等语句后面不要加分号，例如

    错误示例：if(a==0);
正确示例：if(a==0)
错误示例：for(i=0;i<=10;i++);
正确示例：for(i=0;i<=10;i++)

    同时注意for语句中每个语句用分号间隔而不是逗号。
    特别的，在while语句中，while后面无需加分号，而在do-while语句中while后面要加。

  16. 数组当中下标是从0开始的，下标的合理取值范围为[0，数组长度-1]，例如一个规定了10个元素的数组a，a[10]是不存在的。它的元素为a[0]-a[9]。注意在定义数组的时候括号内的是数组内数的个数，在调用数组的时候上限才为个数-1.

  17. 在自定义函数中，return只能返回一个函数值，如果存在多个值需要返回主函数，则应该把自定义函数类型设置为void，对于需要返回的值设置全局变量,void函数中可以存在return，但是return后面不能加任何数值，表示自定义函数到此为止。
